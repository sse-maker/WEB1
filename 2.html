What is CSS? (CSS란 무엇인가?)

Cascading Style Sheets (CSS) is a style sheet language used for describing the presentation of a document written in a markup language such as HTML. CSS is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript.

eliminate duplication in coding is important!

Two methods to insert CSS into the webpage:

  • use <style> tag
  • use syle property

a {
  color:red;
}

  • a -> Selector
  • color:red; -> Declaration
  • color -> Property
  • red -> Value

<style> tag is a grammar of HTML, and contents inside of the <style> tag are CSS.
<style> tag requires additional code, such as '(destination) { }', because it cannot explain to whom it should be assigned.

style="" is property of HTML
Style property is promised to have a CSS effect as a value.
Style property does not have to use a selector(선택자), since it applies the effect to the tag that style property is located.

Two major foundation that rules CSS: Effect and Selector

class="" is property of HTML
Want to find the tag that has class value: use point(.)
The value of the class property can be multiple values, and separated by spacing.
Tag can contain multiple properties and can be jointly controlled by multiple selectors.
(But this is not a good method - tag can be affected by multiple classes)

Influence :ID selector > class selector > tag selector
Want to find the tag that has ID value: use #

Tags that uses full screen: block level element
Tags that uses their own content size: inline element
display: inline; -> change block level element to inline element
display: block; -> change inline element to block level element

    CSS Box Model

  • Specify content size: width, height
  • Interval between content and border: padding
  • Interval between border and border: margin

Determining the volumetricity by treating each HTML tag as a box, and determining volume is a key element in design

<div> tag & <span> tag: tags that are meaningless (just for design purposes)
<div> tag: block level element
<span> tag: inline element
