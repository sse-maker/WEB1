What is JavaScript? (JavaScript란 무엇인가?)

JavaScript, often abbreviated as JS, is a programming language that conforms to the ECMAScript specification. JavaScript is high-level, often just-in-time compiled, and multi-paradigm. It has curly-bracket syntax, dynamic typing, prototype-based object-orientation, and first-class functions.

JavaScript is 'dynamic'
Use <script> tag to inform web browser of javascript code

Properties starting with 'on': called an 'event' in JavaScript
Using events can make the website interact with users.
e.g. onclick property value must have JavaScript code
To have a JavaScript code when an event occurs: onclick
(event: what happens on a web browser)

Console can run JavaScript code on the spot.

Data Types

The latest ECMAScript standard defines seven data types:

Six data types that are primitives:
  • Boolean
  • Null
  • Undefined
  • Number
  • String
  • Symbol (new in ECMAScript 2015)
and Object

When using variables, use the keyword 'var'.

querySelector(): select elements as CSS selectors
(warning: querySelector() selects only the first element corresponding to the selector)
(JavaJcript get element by CSS selector multiple: querySelectorAll())

'===' operator (comparison operator) decides whether the left side value and the right side value has the same value.
Results produced by comparison operators: true or false (these values are booleans)
Boolean data type comes in the if statement.

For code that runs within an event, use a special keyword that is currently promised to point to the tag to which the code now belongs: 'this' keyword (keyword that point to myself)

push(): insert data to the end of the array
Boolean data type comes in the while statement. (runs until false comes in the while statement)
